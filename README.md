# 🚀 Zaibii Deals Engine

Ek advanced **Affiliate Marketing Dashboard** aur **Deals Tracking System** jo real-time mein products manage karta hai aur user clicks ko track karta hai.

## ✨ Features
- **Dynamic Product Management:** Admin panel se bagair code chere naye products (Daraz, Amazon, etc.) add karein.
- **Real-time Tracking:** Har click ka data Firebase Firestore mein save hota hai.
- **Admin Dashboard:** Password protected login jahan total clicks aur estimated earning nazar aati hai.
- **Search Engine:** Users asani se products dhoond sakte hain.
- **Cloud Hosted:** Vercel par live deployment ke liye optimize kiya gaya hai.

## 🛠️ Tech Stack
- **Frontend:** React.js, React Router
- **Backend/Database:** Firebase Firestore
- **Deployment:** Vercel
- **Styling:** Custom CSS (Modern UI)
